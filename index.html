Grundläggande html och css

Vad består en tagg respektive ett attribut av? Hur förhåller de sig till ett element?

En tagg används för att börja och avsluta ett html element i ett html dokument. Ett attribut däremot är instuktioner som används för att klargöra eller modifiera ett element. Ett atttribut defineras i starttaggen.

Vad är skillnaden mellan ett block-element och ett inline-element?

Blockelement är att element i webbläsare ses som rektanglar. Problemet kan däremot bli att hemsidan fortsätter skapa block istället för att behålla meningar i ett block. Då används inline-element, som hjälper till att skapa ett flyt.

Vad är en regel i CSS och vad består den av?

css-regler kan delas upp på två delar. Den första kallas för selektor Selektor används för att bestämma vilka element som ska påverkas av den specifika CSS-regeln. Den andra är en deklaration som över vilka egenskaper som ska förändras samt de värden som de ska sättas till.
